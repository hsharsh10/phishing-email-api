Project Overview

This is an AI-powered phishing email detection system that uses deep learning to analyze email content and determine whether it's a phishing attempt or legitimate communication. The system leverages a trained LSTM (Long Short-Term Memory) neural network model to identify suspicious patterns and characteristics commonly found in phishing emails.

Beautiful, responsive UI with smooth animations

How It Works

The detection process follows these steps:

  • Text Processing: Email content is tokenized and converted into numerical sequences
  • Sequence Padding: Sequences are padded to a fixed length (1000 tokens) for model input
  • LSTM Analysis: The deep learning model analyzes patterns and linguistic features
  • Prediction: Model outputs a probability score (0-1)
  • Classification: Emails with score ≥ 0.5 are classified as PHISHING, otherwise LEGIT

Usage Instructions

Open a terminal and run:

Step 2: Analyze Emails

API Information

Endpoint: POST /predict

Request Body:

  • email_text (string): The email content to analyze

Response:

  • prediction (string): "PHISHING" or "LEGIT"
  • confidence (float): Probability score between 0 and 1

Model Details

The LSTM model (phishing_lstm_model.h5) is trained to recognize patterns in email text that are indicative of phishing attempts, such as urgency tactics, suspicious links, requests for personal information, and other common phishing characteristics. The tokenizer (tokenizer.pkl) converts text into numerical sequences that the model can process.
